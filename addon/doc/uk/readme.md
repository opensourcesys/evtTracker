# Відстежувач подій

* Автори: Джозеф Лі, Тіаго Сеус

Цей додаток виводить інформацію про об’єкти, для яких було викликано події. У режимі журналу налагодження фіксуються такі властивості: тип об’єкта, ім’я, роль, подія, модуль додатка, а також інформація, характерна для API доступності, така як accName для об’єктів IAccessible та Automation Id для об’єктів UIA.

Примітки:

* Цей додаток призначений для розробників та досвідчених користувачів, яким потрібно відстежувати події, що надходять від програм та різних елементів керування.
* Щоб користуватися цим додатком, NVDA має працювати в режимі запису в журнал налагодження (налаштуйте у загальних налаштуваннях/Рівень запису в журнал або перезапустіть програму з увімкненим журналюванням налагодження).
* Можливо, що додатки, завантажені раніше за Відстежувач подій, не передаватимуть події іншим додаткам, включаючи Відстежувач подій. Якщо це станеться, Відстежувач подій не зможе реєструвати події.
* Події обробляються з глобальних плагінів, модулів додатків, перехоплювачів дерева та об’єктів NVDA, у такому порядку.

## Події та інформація про них

Відстежуються та реєструються такі події:

* Маніпуляції з фокусом: отримання фокусу, втрата фокусу, перехід у фокус, перехід на передній план
* Зміни: назва, значення, стан, опис, активна область
* Інші події: сповіщення
* Події UIA: вибір контролера, ефекти перетягування та розміщення, вибір елемента, стан елемента, порушення макета, сповіщення, системне сповіщення, зміна тексту, відкриття підказки, відкриття вікна

Для кожної події буде записано таку інформацію:

* Назва події
* Об’єкт
* Назва об’єкта
* Роль об’єкта
* Значення або стан об’єкта залежно від події
* Модуль програми
* Для об’єктів IAccessible: назва об’єкта, ідентифікатор дочірнього елемента
* Для об’єктів UIA: ідентифікатор автоматизації, назва класу, властивості сповіщення (якщо записується інформація про подію сповіщення), кількість дочірніх елементів для події порушення макета, властивості стану елемента, перетягування та ефект місця призначення, якщо визначено

Ви також можете призначити жест для перегляду подій у списку (меню NVDA/Налаштування/Жести вводу, категорія «Відстежувач подій»). Список зберігає до 100 останніх подій, оброблених до відкриття діалогу.

Якщо ви вважаєте цей додаток корисним, будь ласка, [оцініть його][1] у магазині додатків NVDA.
