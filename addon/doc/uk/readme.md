# Відстежувач подій

* Автори: Джозеф Лі, Тіаго Сеус

Цей додаток показує інформацію про об’єкти, для яких були запущені події. Властивості, що записуються в режимі журналу налагодження, включають тип об’єкта, назву, роль, подію, модуль додатка, а також інформацію, характерну для API доступності, таку як accName для об’єктів IAccessible та Automation Id для об’єктів UIA.

Примітки:

* Цей додаток призначений для розробників та досвідчених користувачів, яким потрібно відстежувати події, що надходять від додатків та різних елементів керування.
* Щоб використовувати цей додаток, NVDA має вести журнал у режимі налагодження (налаштувати можна в загальних налаштуваннях (Приватність та безпека в NVDA 2026.1 та пізніших версіях)/Рівень запису в журнал, або перезапустіть програму з увімкненим журналюванням налагодження).
* Можливо, що додатки, завантажені раніше за Відстежувача подій, не передаватимуть події іншим додаткам, зокрема Відстежувачу подій. У такому разі додаток не зможе реєструвати події.
* Події обробляються з глобальних плагінів, модулів додатків, перехоплювачів дерева та об’єктів NVDA у зазначеному порядку.

## Події та інформація про них

Відстежуються та реєструються такі події:

* Маніпуляції з фокусом: отримання фокусу, втрата фокусу, вхід у режим фокусування, передній план
* Зміни: назва, значення, стан, опис, активна область
* Інші події: сповіщення
* Події UIA: контролер для, ефекти перетягування та місця призначення, вибраний елемент, стан елемента, недійсний макет, повідомлення, системне сповіщення, зміна тексту, відкриття підказки, відкриття вікна

Для кожної події записується така інформація:

* Назва події
* Об’єкт
* Назва об’єкта
* Роль об’єкта
* Значення або стан об’єкта залежно від події
* Модуль додатка
* Для об’єктів IAccessible: ім’я acc, ідентифікатор дочірнього елемента
* Для об’єктів UIA: ідентифікатор автоматизації, назва класу, властивості сповіщення (якщо записується інформація про подію сповіщення), кількість дочірніх елементів для події «макет недійсний», властивості стану елемента, перетягування та ефекту місця призначення, якщо вони визначені

Ви також можете призначити жест для перегляду подій у списку (меню NVDA/Параметри/Жести вводу, категорія «Відстежувач подій»). У списку зберігається до 100 останніх подій, оброблених до відкриття діалогу.

Якщо ви вважаєте цей додаток корисним, будь ласка, [напишіть відгук][1] у магазині додатків NVDA. Список змін, внесених між окремими версіями додатка, дивіться у документі [журнал змін для версій додатка][2].
