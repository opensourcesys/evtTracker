# Відстежувач подій

* Автори: Джозеф Лі, Тіаго Сеус

Цей додаток виводить інформацію про об’єкти, для яких було викликано події. Властивості, що записуються в режимі журналу налагодження, включають тип об’єкта, ім’я, роль, подію, модуль програми, а також інформацію, характерну для API доступності, таку як accName для об’єктів IAccessible та Automation Id для об’єктів UIA.

Примітки:

* Цей додаток призначений для розробників та досвідчених користувачів, яким потрібно відстежувати події, що надходять від програм та різних елементів керування.
* Щоб використовувати цей додаток, NVDA має вести журнал у режимі налагодження (налаштування здійснюється в загальних налаштуваннях (Приватність та безпека в NVDA 2026.1 та пізніших версіях)/рівень журналювання, або перезапустіть програму з увімкненим журналом налагодження).
* Можливо, що додатки, завантажені раніше за Відстежувач подій, не передаватимуть подію іншим додаткам, включаючи Відстежувач подій. Якщо це станеться, додаток не зможе реєструвати події.
* Події обробляються з глобальних плагінів, модулів додатків, перехоплювачів дерева та об’єктів NVDA, у такому порядку.

## Події та інформація про них

Відстежуються та реєструються такі події:

* Керування фокусом: отримання фокусу, втрата фокусу, вхід у фокус, передній план
* Зміни: ім’я, значення, стан, опис, активна область
* Інші події: сповіщення
* Події UIA: контролер для, ефекти перетягування та місця призначення, вибраний елемент, стан елемента, недійсний макет, сповіщення, системне сповіщення, зміна тексту, відкриття підказки, відкриття вікна

Для кожної події буде записано таку інформацію:

* Назва події
* Об’єкт
* Назва об’єкта
* Роль об’єкта
* Значення або стан об’єкта залежно від події
* Модуль програми
* Для об’єктів IAccessible: назва acc, ідентифікатор дочірнього елемента
* Для об’єктів UIA: ідентифікатор автоматизації, назва класу, властивості сповіщення, якщо записується інформація про подію сповіщення, кількість дочірніх елементів для події недійсного макета, властивості для стану елемента, перетягування та ефекту місця призначення, якщо визначено

Ви також можете призначити жест для перегляду подій у списку (меню NVDA/Параметри/Жести вводу, категорія «Відстежувач подій»). Список зберігає до 100 останніх подій, оброблених до відкриття діалогового вікна.

Якщо ви вважаєте цей додаток корисним, будь ласка, [оцініть його][1] у магазині додатків NVDA.
